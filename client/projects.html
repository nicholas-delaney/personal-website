Blackjack Reinforcement Learning AI

An app that trains an agent to play blackjack using the Reinforcement Learning Q-Learning algorithm. The agent learns to either "Hit" or "Stick" based on the current state: the sum of the agent's hand and whether or not the agent has an ace in their hand. The app lets you perform iterations of the Q-learning algorithm and displays which action the agent thinks it should take at any given state. This project uses artificial intelligence, JavaScript, HTML, CSS, Node.js and Express.
